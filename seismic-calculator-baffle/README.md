# Seismic Calculator Baffle

Vue 3 + Tailwind CSS calculator for seismic bracing calculations for suspended baffle ceilings.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist/` directory for deployment to JSDeliver.

## Deployment

See `DEPLOYMENT.md` for instructions on deploying to GitHub and JSDeliver.
